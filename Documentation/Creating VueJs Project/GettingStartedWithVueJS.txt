1. Open Command Prompt and Run npm install -g @vue/cli (I think it's a once off thing)
2. To create a project -> vue create hello-world(Your Project Name) and answer all the setup questions that follows.
3. Open project in VS Code -> go to externtions and search for (@id:octref.vetur) and install it
